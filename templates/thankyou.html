2020 January

Case Study Cyber-physical production systems - Post-processing of 3D printed objects/Additive manufacturing.

Additive manufacturing/ 3d printing is one of the production technologies we can use in cyber physical production. 3d printing is used to produce complex parts with individual customisation in a very fast way. 3d CAD models designed in CAD tools are sliced into layer digitally and printed layer by layer using a 3d printer. By the design modification and adjusting part orientation we can minimise supporting structure. If the situation of avoiding support structure is inevitable or the surface is not smooth, post processing will be required. Different methods for post processing are analysed and categorised based on models and the AM technologies used in 3d printer. 3d printer types, technologies , material used and 3d models influences end products surface smoothness.

Predictive analysis is one of the best tool for optimum production planning. Machine learning algorithms and deep learning are powerful way to predict data based on relevent data base. Regression as well as classification are discussed in the case study. Regression analysis layer by layer is done using Matlab and tensor-flow. The model is used to analyse the trend of sensor data of machines to predict next possible maintenance time. By using ML the downtime of the machine in an industry is predicted. The modelling of the system was the key point in the case study.

Projects

Seizure detecting smart Band – an Embedded C innovation

2018

Designed a preclinical demonstrator for real-time detection of seizure based on various body sensors. The Embedded System consist of a microprocessor (ARM Cortex M4) along with heart rate sensor, body temperature sensor and ectodermal sensor acts independently without the help of any peripheral device and inform the dependent during the activity of seizure. It also collects details as a log during the seizure which assists the doctor during patients diagnosis. The band is compact and specially designed for epilepsy patients. The Data collected using the sensors will be used to check the trend during the siezure and predict the future incident to be happened by creating individualised algorithm.
